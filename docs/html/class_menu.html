Detailed Description

a Menu that bases its implementation on a stack

Every menu has two methods: show() - responsible for the "view" in the console, provides useful messages to the user or important info in text form getNextMenu() - responsible for the behaviour aspect of the program, receives inputs and returns the new path in form of a pointer
